# Student Assistant

Приложение для студентов и школьников, которое помогает организовать учебу: вести расписание занятий и отслеживать
домашние задания. Простое, удобное и с открытым исходным кодом.

---

## Основные функции

### Расписание
- Создание и редактирование расписания на семестр
- Добавление занятий: предмет, преподаватель, аудитория, время
- Поддержка повторений по дням недели
- Управление несколькими расписаниями

### Домашние задания
- Добавление заданий к предметам
- Указание дедлайна и описания
- Привязка к предметам из расписания

### Настройки
- Установка времени начала первого занятия
- Настройка длительности уроков и перерывов
- Расширенный выбор повторений (гибкая настройка недель)

---

## Архитектура проекта

Проект разработан с использованием **модульной архитектуры** на Kotlin, что обеспечивает гибкость и легкость поддержки.

### Структура модулей
```
features/
├── schedule/       — Модуль расписания
├── homeworks/      — Модуль домашних заданий
├── settings/       — Модуль настроек
└── repository/     — Общий репозиторий данных

core/
├── strings/        — Строковые ресурсы
└── ...             — Общие компоненты (UI, навигация, архитектура)
```

- Написано на **Kotlin**, с использованием современных практик Android-разработки.
- Поддерживает **Material Design**.
